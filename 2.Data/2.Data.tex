\begin{frame}{八、误差和错误}
\protect\hypertarget{ux516bux8befux5deeux548cux9519ux8bef}{}
\begin{itemize}
\tightlist
\item
  误差：也称为试验误差，是指观测值偏离真值的差异，分为随机误差和系统误差。

  \begin{itemize}
  \tightlist
  \item
    随机误差：由于试验中许多无法控制的偶然因素所造成的试验结果与真实值之间的差异，是\textbf{不可避免}的。
  \item
    系统误差：由于试验处理以外的其他条件明显不一致所产生的带有倾向性的或定向性的偏差。
  \end{itemize}
\item
  错误：在实验过程中，人为因素引起的差错。
\end{itemize}
\end{frame}
